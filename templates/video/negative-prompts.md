# 负面提示词库

> 按镜头类型选用组合，直接粘贴。

## 通用基础包

```
跳切，硬切，镜头瞬切，景别突跳，变形的手指，多指少指，模糊五官，角色融合，悬浮漂移，卡顿瞬移，肢体抽搐，背景音乐，水印
```

## 双角色同框追加

```
左右换位，位置对调，角色交叉走位，镜像翻转，两人互换边
```

## 锁帧衔接追加

```
结尾与尾帧不符，结尾跑飞，在尾帧上继续乱动变形
```

## 含文字画面追加（海报/标题板背景）

```
海报文字缺失，海报文字乱码，画面内生成乱码字，彩色假英文字母
```

## 强光/特效段追加

```
强光糊脸，角色被光吞没，光效闪烁，光效中出现文字
```
