# 一致性硬约束（视频生成的六条铁律）

> 这些规则每条都对应一类真实翻车。写提示词时按需成段复制。

## 1. 锁帧衔接（多镜头拼片的基础）

平台支持关键帧锁定时：起始关键帧设**首帧锁定**、结尾关键帧设**尾帧锁定**，并在提示词里写明：

```
结尾连续过渡后精确回到尾帧构图并完全静止定格，与尾帧像素级一致，给足约 2 秒落帧定格时间；严禁结尾跑飞、与尾帧不符、在尾帧上继续乱动变形。
```

含正式文字的画面（海报、标题板）只能靠锁帧保住——模型直接画文字必崩。

## 2. 站位锁死（双角色同框必写）

```
全程[A]恒在画面左、[B]恒在画面右，左右绝对不变；严禁左右调换、交叉走位、镜像翻转、绕到对方另一侧；机位正面平视，绝不越过两人中轴线，绝不绕行；推拉只变景别不变左右。
```

## 3. 一镜连续（防硬切）

```
全片从首帧起一条不间断连续长镜头，景别只做平滑匀速渐变推拉；严禁跳切、硬切、镜头瞬切、画面突跳、景别突跳。
```

## 4. 运动控制（防漂移抽搐）

```
肢体与手部轨迹连续无断点；双脚踏地重心稳定；严禁悬浮漂移、瞬移、关节死锁、抽搐、只动嘴不动身。
```

## 5. 特效专控（强光/粒子段必写）

```
[特效]柔和放射、光点细腻有层次，不糊脸、不抢角色；强光中角色轮廓清晰、五官完整、身份不变形不融化；特效运动连贯不闪烁，绝不在光效里生成乱码字或假英文字母。
```

## 6. 声音纪律

```
全程禁止任何背景音乐与旋律垫底；只保留环境音效与角色对白配音；音效与口型动作节奏逐帧对齐。
```

BGM 后期统一铺连续底乐；各段自生成的音乐风格无法衔接，且会盖住对白。
