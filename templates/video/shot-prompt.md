# 单镜头视频提示词模板（逐秒时间轴）

> 用途：以定版图像为起点，把一个镜头写成模型可执行的视频提示词。
> 前置：先用 `templates/image/` 完成图像定版——图生视频远比文生视频可控。

```
[起止设定]：以附带图像为首帧[如平台支持：锁定首帧/尾帧]。总时长[必须为整数秒，如 8 秒]。
[镜头运动]：[一条连续镜头；运镜宁朴勿炫：正面直推/直拉/固定机位；写明"严禁跳切、硬切、景别突跳"]。
[主体动作时间轴]：
  0–X 秒：[谁做什么可见动作，幅度与方向]。[有对白则标：口型对应"……"；无对白空镜标：无口型动作]
  X–Y 秒：[下一拍动作]。
  末 1.5–2 秒：[收尾定格描述，便于与下一镜衔接]。
[主体锁定]：[角色外观关键特征逐项重复；多角色写明"A 恒在画面左、B 恒在画面右，全程左右绝对不变"]。
[声音]：[建议：全程禁止背景音乐，只留环境音效与对白——BGM 后期统一铺，否则各段自生成的音乐无法衔接]。
[风格与画质]：[渲染风格]，[光照与色调]，[画幅]。
[负面约束]：见 negative-prompts.md，按镜头类型选用。
```

## 要点

- **时间轴卡秒**：每拍标"几秒到几秒"，对白起止卡进秒数——口型与配音对齐的前提。
- **站位锁死**：双角色同框必写"谁左谁右、全程不变"，并禁越轴运镜（越过中轴线=镜像翻转元凶）。
- **一镜一任务**：一个镜头只完成一个叙事动作；复杂调度拆成多镜后期拼接。
- **哪一拍崩就单独重生成那一拍**，不要整段重来。
- **省字数**：部分平台把空格换行计入字数且计数偏严；删装饰符号，省下的字写更细的动作描述。
