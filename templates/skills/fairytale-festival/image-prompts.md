# 童话节 · 生图方案

## 1. 节日舞台氛围（主视觉/大场景）

```
画面主体：[剧场/舞台/图书馆大厅]，[台上主体]，台下坐满卡通小朋友与家长。
氛围要素（选用）：暖金色氛围光；漫天细腻金光点；发光的立体书；书页化作飞鸟；小绿芽从书中萌发；彩色立体翻开书籍拼成的[球体/拱门/阶梯]。
光线与色调：暖金主调（黄昏金/舞台金二选一），柔和体积光，轮廓光勾边。
风格：高端3D卡通动画电影长片质感，皮克斯式角色塑形，院线级CG，电影级全局光照，8K精细渲染纯净无噪点。
画幅：16:9。
负面约束：无文字，无水印，画面内不生成任何标语馆名（正式文字后期添加）。
```

## 2. 双角色同框（主持/互动）

```
画面主体：[角色A：外观占位]在画面左，[角色B：外观占位]在画面右，[可见动作：挥手/鞠躬/展臂/互相夸赞]。
站位：A恒左B恒右；两角色比例关系[如：A身高约为B的1.8倍]。
表情与姿态：[具体到眉眼与手势，如：眉眼含笑、右手向观众方向大方一引]。
环境：[舞台/书架前/阅读区]，背景[观众/童书陈列/节日装饰]。
光线风格画幅：同方案1。
负面约束：变形的手指，多指少指，模糊五官，角色融合，两角色特征互串。
```

## 3. 阅读意象空镜（转场/升华）

```
画面主体（任选其一展开）：一本徐徐翻开的发光立体书；种子破土长成书页树；书页化作青鸟群飞向城市天际线；一排亮灯的书房窗口。
运动感提示：为后续图生视频预留动势（如"书页微微掀起""光点悬浮"）。
光线：[黄昏暖金/夜晚蓝金对比/清晨柔白]。
风格画幅负面约束：同方案1。
```

## 要点

- 节日感三件套：暖金光 + 细腻光点 + 书籍意象，比堆砌装饰元素更稳定。
- 儿童友好审美：圆润造型、明快饱和但不刺眼的配色、人物眼神有神。
- 同一活动的所有图共用同一段"风格行"文字，整体视觉才统一。
