# 童话节 · 生视频方案

## 0. 全片结构原则（先于任何单镜头）

- **连续音轨优先**：先录整段旁白/对白（VO），按语义点切分镜；不要让对白因分镜切换而断裂。
- **口型镜头显式绑定**：角色正面/中近景说话镜头，提示词写明 `口型对应"……（台词）"`；蒙太奇与空镜写 `旁白继续，无口型动作`。
- **文字后期加**：AI 生成的图与视频中避免直接生成中文、数字、标签、机构名；正式文字全部在剪辑软件中后期添加。
- **音效剪辑点**：建议在剪辑点前 0.2–0.4 秒提前进入下一镜音效，拼接更顺。

## 1. 双角色主持镜头（口型段）

```
以附带定版图为首帧[平台支持则锁首帧]。总时长[整数秒]。
镜头：正面平视，中景同框，机位[固定/极缓慢匀速直线微推]，全程不切。
站位铁律：[A]恒左[B]恒右（完整段落见 templates/video/continuity-rules.md 第2条）。
时间轴：
  0–X秒：[A动作]。对白[A]：“[台词]”（口型对应）。音效[观众反应]。
  X–Y秒：[B接话动作]。对白[B]：“[台词]”（口型对应）。音效[观众反应]。
声音：全程禁止背景音乐，只留环境音效、欢呼掌声与对白配音，逐帧对齐口型。
角色锁线：[A外观逐项占位]；[B外观逐项占位]；手指恒五根，身份独立绝不互相变形或面部互串。
背景锁线：[环境描述]与首帧一致全程不变。
负面约束：通用基础包＋双角色同框追加（见 negative-prompts.md）。
```

## 2. 收尾升华镜头（特效+定格）

```
承接上段的收尾升华，一镜到底连续推拉，锁首尾帧。总时长[整数秒]。
[中段时间轴：角色展臂/踮脚/笑容上扬，金色光晕自下而上升起，空气泛细微金尘]
[高潮拍]：画面中心金色光芒柔和放射由内向外绽放，漫天细腻金光点，礼花般层层铺满舞台。
特效专控：金光柔和放射不糊脸不抢角色；强光中角色轮廓清晰五官完整；绝不在光效里生成乱码字。
末2秒：镜头光效收稳，平滑过渡到与尾帧完全一致构图后全部静止，像素级锁死定格，给足落帧时间无缝衔接下段。
声音：无对白段承接前句余韵；音效[一记温暖魔法闪光叮＋掌声欢呼渐弱]。
```

## 3. 阅读意象空镜（旁白段）

```
以定版空镜图为首帧。总时长[整数秒]。旁白继续，无口型动作。
运动：单一方向的缓慢运动（书页掀起/光点上升/镜头缓推），宁朴勿炫。
[意象展开：见 image-prompts.md 方案3]
声音：只留环境音效，为旁白留出干净声场。
```

## 排查清单（每段生成后过一遍）

首帧＝你的定版图？→ 有无硬切？→ 站位有无翻转？→ 背景文字有无丢失/乱码？→ 特效段有无糊脸？→ 结尾是否干净定格在尾帧？哪一拍崩就单独重生成那一拍，别整段重来。
