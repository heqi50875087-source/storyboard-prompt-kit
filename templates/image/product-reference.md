# 产品/道具参考图模板（四视角一致性）

> 用途：为反复出现的道具、产品、标志物生成一致性参考图（如活动主视觉里的发光书、吉祥物手持道具）。

```
产品参考图 —— 中性灰背景，四视角：

[视角 1 — 正面，四分之三] [产品 + 颜色/质感] 的正面四分之三视角，从上到下完整可见。[正面主要特征]。
[视角 2 — 背面，正视] 同一 [产品] 的完整背面视角，正后方直视。标志性特征：[背面核心特征]。[负面约束——背面绝不应出现的内容]。
[视角 3 — 正面特写] [某区域] 特写，正面直视，[主要细节] 锐利清晰，[材质/纹理/交界过渡] 细节精细。
[视角 4 — 侧面] 完整 [左/右] 侧面特写，展示边缘剖面与厚度过渡。

灯光与呈现：干净的影棚灯光——左上方柔和主光，右侧轻微补光。四个视角之间产品身份、比例、颜色与细节保持一致。无文字、无水印、无多余物体、无背景环境。风格：[写实产品摄影 / 与项目统一的渲染风格]
```

## 元提示词（让大模型帮你填模板）

把本模板连同这句话一起丢给任意大语言模型：

```
把这个框架改写成一份用于 [插入产品/道具] 的参考图提示词。
```
