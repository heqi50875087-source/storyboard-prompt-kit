# 分镜生图模板

> 用途：把一个分镜意图变成稳定出图的生图提示词。核心原则：**抽象形容词是不稳定输出的主要来源**——"自然""梦幻"这类词必须改写为可见动作与空间关系。

```
[画面主体]：[谁/什么]在[哪里]做[什么可见动作]。
[主体锁定]：[角色外观关键特征逐项重复——模型没有记忆，每个镜头都要重新写全]。
[构图与镜头]：[景别：远/全/中/近/特写]，[机位：平视/俯拍/仰拍]，[主体在画面中的位置]。
[光线与色调]：[光源方向与性质]，[时段]，[主色调与氛围，用具体色名不用情绪词]。
[风格]：[渲染风格 + 质感等级，如：高端3D卡通动画电影质感，电影级全局光照，8K精细渲染]。
[画幅]：[16:9 / 9:16 / 1:1]。
[负面约束]：无文字，无水印，无多余人物，[按需补充：变形的手指，多指少指，模糊五官]。
```

## 示例（占位角色）

```
画面主体：一位戴黑框眼镜的年轻馆员和一只圆头大眼的小蜜蜂吉祥物并肩站在儿童剧场舞台中央，向观众挥手。
主体锁定：馆员[发型/服装/标志物]；吉祥物[体色/头部特征/翅膀/服饰]。
构图与镜头：中景，正面平视，两人居中，馆员在画面左、吉祥物在画面右。
光线与色调：舞台顶光加暖金色氛围光，橙金主调，节日感。
风格：高端3D卡通动画电影质感，皮克斯式角色塑形，电影级全局光照，8K精细渲染。
画幅：16:9。
负面约束：无文字，无水印，无多余人物，变形的手指，模糊五官。
```

## 要点

- **数字要具象化**：不写"很多书"，写"约三十本彩色童书堆成弧形"。
- **一图一重点**：一张图只讲一个视觉重点；要素堆太多，模型每个都画不好。
- **多版本比选**：同一分镜生成 2–3 版（可变化光线时段/核心意象），按叙事连贯性人工定版——图像层的筛选劳动是视频质量的保险。
