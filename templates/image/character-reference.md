# 角色参考图模板（四视角一致性）

> 用途：为每个反复出现的角色生成一份"风格圣经"参考图，后续所有镜头引用它保持角色一致。
> 用法：替换方括号字段后投喂文生图模型；同一角色的参考图尽量在同一会话（可选同一 seed）内生成。

```
展示同一个 [角色描述：年龄气质、发型发色、五官特点、服装与配色、标志性物件]

角色参考图 —— 中性灰背景，四视角：

[视角 1 — 全身，正面] 该角色的全身正面四分之三视角，从头到脚完整可见。
[视角 2 — 全身，背面] 同一角色的全身背面视角，正后方直视，从头到脚完整可见。
[视角 3 — 正面特写] 头肩部特写，正面直视；皮肤质感、配饰、服装表面细节锐利清晰。
[视角 4 — 侧面特写] 头肩部特写，左侧 90 度正侧面视角，颈部与上肩可见。

灯光与呈现：干净的影棚灯光——左上方柔和主光，右侧轻微补光。四个视角之间角色身份、比例与服装细节保持一致。无文字、无水印、无多余人物、无背景环境。风格：[插入所需风格，如：高端3D卡通动画电影质感 / 写实电影 / 2D手绘]
```

## 要点

- 角色描述控制在 80–150 字：太短锁不住身份，太长模型抓不住重点。
- 多角色项目：每个角色一份独立参考图，描述用可区分的关键词（避免"都是短发小孩"这类会互相污染的写法）。
- 后续镜头引用写法：`使用附带的[角色]参考图，保持形象、比例、颜色与服装细节与参考图完全一致。`
