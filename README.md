# Storyboard Prompt Kit · 分镜提示词工具

一款面向 AI 视频生产的轻量提示词工具：把「分镜意图」一键组装成可直接投喂文生图 / 图生视频模型的结构化提示词。

源自一线公共文化机构「零专项经费、单人执行」的 AI 视频生产实践——一支约 8 分钟的开幕式宣传片、数千张生成图像与数十个视频镜头的试错沉淀。规则不是理论推演，是被失败链逼出来的。

## 它解决什么问题

生成式模型的不稳定输出大多来自三件事：**抽象形容词**（"梦幻""有节奏感"无法转化为可控画面）、**角色漂移**（模型没有记忆，每个镜头都可能长得不一样）、**镜头失控**（乱切、越轴、镜像翻转）。本工具把应对这三件事的规则固化成模板：

- **生图模板**：角色 / 产品参考图（四视角一致性）、分镜生图（主体锁定 + 风格 + 构图 + 负面约束）
- **生视频模板**：单镜头提示词（动作 / 镜头 / 逐秒时间轴 / 口型绑定）、一致性硬约束（锁首尾帧、站位锁死、一镜连续、特效控制）、负面提示词库
- **可选 Skill**：`fairytale-festival` 童话节 / 少儿节日活动方案（生图 + 生视频成套，角色外观已做占位符化处理，替换为你自己的角色即可）
- **参考卡库**：`templates/skills/reference-cards/` 六张技法卡，供直接阅读取用——分镜导演、生图角色一致性、电影化提示词、视频分镜硬技巧、运镜与 VFX、GPT-Image 生图技法（源自 Higgsfield 官方预设与多平台实践的调研甄别）

## 快速开始

### 方式一：零依赖干跑（不调用任何模型，推荐首次使用）

```bash
python3 agent/generate.py --type image --intent "一只戴眼镜的小熊在图书馆里给小朋友讲绘本，暖色灯光" --dry-run
```

输出组装好的完整提示词包，复制粘贴到任何文生图 / 视频平台即可。

### 方式二：接入模型自动生成（本地或云端，OpenAI 兼容接口通用）

```bash
cp agent/config.example.json agent/config.json   # 填入你的 endpoint / model / api_key
python3 agent/generate.py --type video --intent "两个角色在舞台上向观众鞠躬致谢，金色灯光，12秒" 
```

- **本地部署**：Ollama / LM Studio 等本地服务默认暴露 OpenAI 兼容接口（如 `http://localhost:11434/v1`），`api_key` 随意填
- **云端 API**：任何 OpenAI 兼容服务（DeepSeek、Moonshot、通义等）填对应 endpoint 与 key

加载童话节 Skill：

```bash
python3 agent/generate.py --type video --skill fairytale-festival --intent "开幕式收尾，两个角色感谢观众" --dry-run
```

## 目录

```
templates/image/     生图模板（角色参考图 / 产品参考图 / 分镜生图）
templates/video/     生视频模板（镜头提示词 / 一致性硬约束 / 负面提示词库）
templates/skills/    可选场景方案（fairytale-festival 童话节）
agent/generate.py    组装器（纯标准库，零依赖）
examples/            端到端示例
```

## 三条使用原则（血泪经验，先读再用）

1. **图像先行**：先批量生图、人工定版，再以定版图为起点做视频——不确定性每在图像层多消化一分，视频层的返工就少一分。
2. **免费试错、付费出片**：用低成本通道反复验证提示词，稳定后再进付费生成；未经验证的提示词不进付费环节。
3. **文字后期加**：不要让模型直接生成正式文字（标题、馆名、标语），全部在剪辑软件中后期添加——模型画文字必崩。
